# my-agent-app

A clean reference architecture for an agent application.

## Run

```bash
uv run python -m my_agent_app --text "Please verify your account now"
```

## Layout

```text
my-agent-app/
├── pyproject.toml
├── uv.lock
├── README.md
├── LICENSE
├── .gitignore
├── .env.example
├── src/my_agent_app/
├── examples/
├── tests/
├── scripts/
├── docs/
└── configs/
```
