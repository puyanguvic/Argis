# Feature maturity

High-level maturity notes (intended for readers of the docs site).

| Area | Status | Notes |
|---|---|---|
| Core detection pipeline | Beta | Evidence-first, deterministic tools by default. |
| JSONL record + replay | Beta | Enabled only via `--record`. |
| Gmail/IMAP connectors | Experimental | Connector input kind is supported; CLI is JSON-only. |
| Provider-backed models | Experimental | Optional adapters exist; policy remains deterministic. |
